    5.1: Flattening

    Given this array of arrays: [[1, 2, 3], [4, 5], [6]] Create this array: [1, 2, 3, 4, 5, 6]

    Result: 

                      let arrays = [[1, 2, 3], [4, 5], [6]];
                  // Result required: [1, 2, 3, 4, 5, 6]

                  // Your code here.

                  const flatten = (array => {
                    let newarray = [];
                    array.forEach(subarray => {
                        subarray.forEach(item => {
                            newarray.push(item);
                        });
                    });
                    newarray.sort();
                    return newarray;
                });
