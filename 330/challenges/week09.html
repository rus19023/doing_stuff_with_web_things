Challenge 2 Proposal

February 28, 2022

The Problem

Many people who want to study the gospel of Jesus Christ don't have easy access to tools that help them learn the gospel principles quickly. Some of these people are LDS, and they remember the Scripture Chase from Seminary, but there is no such thing once they are out of Seminary.

The Solution

Scripture chase, in digital form! I know it already exists, but as a paid option. I don't believe in making money from anything Gospel-related (priestcraft comes to mind?), so my app will be free. Not even any ads! App will have disclaimer that it is not an official church app with appropriate links to official webpages.

App Title: Scripture Chase


App Description:

Using quiz style or flashcards that flip (in practice or quiz mode), and possibly scrambled scriptures to put in correct order, anyone can study and memorize King James version Bible and other LDS scriptures that teach basic gospel doctrine, just like in LDS Seminary classes.

The app will start with the user viewing a home screen, with "What do you want to study?" followed by a list of the 4 units with thumbnail images and options to select all options. The bottom nav buttons will have text and icons: HOME, QUIZ, FLIPCARD, CUSTOMIZE and SETTINGS.

If they wish to create and save preferences, they tap the CUSTOMIZE button, and a login screen asking for username and password show. If they don't have an account, in the login screen they can create one, after which they will be asked to login. Once logged in, the Dashboard screen will show, with bottom nav buttons of HOME, QUIZ, FLASHCARDS, XXX and LOGOUT.

If they wish to return to the home screen, they tap the HOME button.

Then it will go to a main menu where they can start recording a journal entry, have it sent to the API to be transcribed, receive the journal transcription text, save it to a file, store that file in their journal file system. If time permits, the app will also have a delete function for files, flag function for special entries, printing options and off-site storage options (such as cloud storage like OneDrive, computer hard drive, usb etc.)..

App Technology:

  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • Bootstrap
  • Github API JSON file
  • Firebase

App Goals:

Scripture data comes from API/external JSON source.

User can sign up and log in to save their preferences with filtered, verified and sanitized inputs.

User can save their filtered, verified and sanitized preferences to Firebase.

List of scripture units. Once a unit or units are chosen, a quiz or flashcard base is prepared for the user.

Core Features:

  • Quiz on all or any combination or only one of the four units:
    • Book of Mormon
    • Old Testament
    • New Testament
    • Doctrine and Covenants/Pearl of Great Price
  • Flash cards that flip, with a scripture-related image (if available) and a clue. Start with either the reference to recite the passage or the passage to memorize the scripture text.
  • Track learning progress and stats.
  • Retry missed flashcards or quiz questions.
  • Bottom navigation bar for easy finger reach.
  • Leaderboard.
  • Spanish language.
  • User theme options, including dark/light.
  • Share progress/accomplishments on social media.

Stretch features

  • Put words from the scrambled scripture in proper order to help memorize the scripture passage.
  • Drag and drop fill in the blanks.
  • Add custom topics. This feature will require the use of a database to store the new topics.
  • Add custom questions and answers. This feature will require the use of a database to store the new questions and answers.
  • Google translate into multiple languages.
  • Voice to text, speak the scripture to see if it is memorized correctly.
  • Text to voice, listen to the scripture to help memorize it correctly.
